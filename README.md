## GUI vs CLI
1. GUI (Graphic User Interface): 그래픽을 통해 사용자와 컴퓨터가 상호 작용하는 방식
2. CLI (Command Line Interface): 터미널을 통해 사용자와 컴퓨터가 상호 작용하는 방식

`GUI가 사용하기 쉽지만, CLI는 GUI로는 불가능한, 많은 기능을 사용할 수 있습니다.`

## 터미널 명령어
`touch`
- 파일 생성 명령어

`mkdir`
- make directory
- 새 폴더 생성 명령어

`ls`
- list segments
- 현재 작업중인 디렉토리의 폴더/파일 목록 보여주는 명령어

`mv`
- move
- 폴더/파일 다른 폴더로 이동/변경 명령어

`cd`
- change directory
- 현재 작업 중인 디렉토리 변경하는 명령어

`rm`
- remove
- 폴더/파일 지우는 명령어

`start`
- 폴더/파일 여는 명령어

#### 주요 문법
1. 제목
   - `#`을 사용
   - 갯수가 많을수록 작아짐
2. 목록
   - 순서가 없는 목록
     - `- * +`
   - 순서가 있는 목록
     - `1. 2. 3.` 
3. 강조
   - 기울임: `*글자*`
   - 굵게: `**글자**`
   - 취소: `~~글자~~`
4. 코드
   - 인라인 코드: `inline code`처럼 백틱 사용
   - 블록 코드: ```python처럼 3번 입력하고 코드 종류 작성
5. 링크
   - 클릭하면 해당 주소로 이동
   - [표시할 글자](이동할 주소) 형태 작성
6. 이미지
   - 이미지 삽입
   - ![대체 텍스트](이미지 주소) 형태 작성
7. 인용
   - 주석/인용 문구 표현
   - `>` 사용, 갯수에 따라 중첩 가능
